Step 1: install required libraries found in requirements.txt
    pip install -r requirements.txt               (globally or into a virtual enviroment)

Step 2: jupyter notebooks 
    Now, when all requirements are installed, feel free to 
    run the notebooks in any way that suits you. Alternatively, 
    all content from the last run are available. 

Step 3: train the VAE 
    Run P3_1_vae_template.py found in folder p3 

Step 4: The toy example of the spiral distribution 
    Run P3_2_spiral_score_template.py found in p3 

Step 5: Score-based training
    Run P3_3_latent_diffusion_template.py found in p3 

Step 6: Generate numbers
    Run P3_4_generate.py found in p3
